Trainingsplan

Tag 1: Oberkörper-Training

Aufwärmen: 5 Minuten auf der Gymnastikmatte, Dehnung des oberen Rückens, Schultern und Brust mit Stretchbändern.

Übung Sätze Wiederholungen
Kurzhantel Bankdrücken 3 8-10
Kurzhantel Fliegende 3 10-12
Kurzhantel Rudern 3 8-10
Kurzhantel Bizepscurls 3 10-12
Trizeps-Extensions mit Stretchband 3 12-15
Klimmzüge 3 so viele wie möglich

Tag 2: Unterkörper-Training

Aufwärmen: 5 Minuten auf der Gymnastikmatte, Dehnung der Beine mit Stretchbändern.

Übung Sätze Wiederholungen
Kurzhantel-Kniebeugen 3 8-10
Ausfallschritte mit Kurzhanteln 3 10-12 pro Bein
Kurzhantel-Sumo-Kniebeugen 3 10-12
Kurzhantel-Rumänisches Kreuzheben 3 10-12
Wadenheben auf der Kante mit Kurzhanteln 3 12-15

Tag 3: Ganzkörper-Training

Aufwärmen: 5 Minuten auf der Gymnastikmatte, Dehnung des ganzen Körpers mit Stretchbändern.

Übung Sätze Wiederholungen
Kurzhantel-Kreuzheben 3 8-10
Kurzhantel Schulterdrücken 3 10-12
Kurzhantel Ausfallschritte mit Überkopf-Presse 3 10-12 pro Bein
Kurzhantel Bizepscurls 3 10-12
Trizeps-Extensions mit Stretchband 3 12-15
Klimmzüge 3 so viele wie möglich
